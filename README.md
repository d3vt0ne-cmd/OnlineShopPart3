Часть 1: Создание базовой структуры проекта
**Цель:** Построить фундамент приложения — клиент-серверную архитектуру с разделением фронтенда и бэкенда.

Выполнено:
- Создание Laravel-проекта с PostgreSQL
- Настройка React-приложения на Vite
- Реализация маршрутизации (React Router)
- Защита маршрутов (AuthLayout, UserLayout)
- Глобальное состояние через Context API
- Базовый дизайн страниц (главная, логин, регистрация, профиль)
- Навигационная панель с условным отображением
