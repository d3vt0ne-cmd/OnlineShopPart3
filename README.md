Часть 3: Реализация каталога товаров (меню кофе)
**Цель:** Подключить базу данных для хранения товаров и вывести динамический каталог кофе на фронтенде.

Выполнено:
- Создание моделей `SizeCoffee` и `Coffee` с отношением belongsTo
- Миграции и seeders для заполнения тестовыми данными
- API-ресурсы для форматированного JSON-ответа
- Контроллер CoffeeController с пагинацией
- Роут /api/coffee
- Страница меню (/menu/coffee) с динамической подгрузкой товаров
- Вывод карточек со всеми полями (название, описание, цена, размер, изображение)
- Визуальное обозначение недоступных товаров (available = false)
- Изменения в базе (pgAdmin) сразу отражаются на сайте
